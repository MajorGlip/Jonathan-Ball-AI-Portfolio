# Computer Vision — ITAI 1378

## Course Overview

This course introduced the foundations of computer vision — the field of enabling machines to interpret and understand visual information from the world. Topics covered included image preprocessing, feature extraction, edge detection, object detection, and convolutional neural networks (CNNs).

## Technologies Used

- Python 3.x
- OpenCV (cv2)
- TensorFlow 2.x / Keras
- YOLOv5
- NumPy, Matplotlib

## Projects

### [Project 1 — Pharmaceutical Pill Image Classifier](./Project1-Pill-Classifier/)
A CNN model trained to classify different pill types from images. Inspired by my background as a pharmacy technician, this project explores how AI can reduce medication errors and support pharmacy workflows.

### [Project 2 — Real-Time Object Detection System](./Project2-Object-Detection/)
A real-time object detection pipeline built with YOLOv5 and OpenCV, capable of identifying and labeling multiple object classes from live video or static images using the COCO dataset.

### [Project 3 — Lane Detection with Edge Analysis](./Project3-Lane-Detection/)
A lane detection system using classical computer vision techniques including Canny edge detection and the Hough Line Transform, applied to dashcam-style road footage to identify and highlight lane markings.
